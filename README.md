 Game Instructions

1. Generate a Number: A random number between 1 and 20 will be generated.
2. Make a Guess: Enter your guess.
   - If your guess is too low, the game will display "Too low."
   - If your guess is too high, the game will display "Too high."
3. Score System: Each incorrect guess reduces your score by 1.
4. Winning: Guess the number with the fewest possible inputs. You have a total of 8 guesses (scores) to find the correct number.
5. Game Over: If you guess the number correctly within 8 tries, you win. Otherwise, you lose.
6. Play Again: Click the "Play Again" button to start a new game.
7. High Score: The high score will be displayed.

Good luck and try to guess the number in the fewest attempts!
